## Forensic Audit
Full audit conducted. 11 issues found and ALL fixed:
- **CRITICAL**: Admin middleware queried ClickHouse instead of PostgreSQL (wrong database)
- **CRITICAL**: SQL injection in admin middleware (string interpolation)
- **CRITICAL**: Single-origin CORS
- **HIGH**: No rate limiting, no audit logging, no impersonation return path
- **MEDIUM**: Console.log leak, placeholder secret key, avatar memory leak, missing DB types
